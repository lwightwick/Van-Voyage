
# Van Voyage

A van rental application allowing users to place a booking for a select number of nights.

## Built with
- React.js 
- node.js Express
- CSS
- AWS S3

## Installation
1. Clone repo

```bash
git clone https://github.com/lwightwick/Van-Voyage.git
```

2. Install NPM packages   
```bash
  npm install
```
